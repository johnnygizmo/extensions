Introduction

The Hinge Extrude add-on provides a powerful mesh editing tool that allows you to extrude selected faces while rotating them around an active edge. This is particularly useful for creating hinged geometry, circular extrusions, and complex mechanical parts.

Author: Johnny Matthews
Version: 1.0.5
Blender Version: 4.2.0 or later
Category: Mesh

Installation

This extension can be installed directly from Blender's Extensions repository:

  1. Open Blender 4.2 or later
  2. Go to Edit → Preferences → Get Extensions
  3. Search for "JohnnyGizmo Hinge Extrude"
  4. Click Install

Usage

Prerequisites

To use the Hinge Extrude tool, you must:

  • Be in Edit Mode on a mesh object
  • Have both Edge and Face select modes enabled
  • Have at least one face selected
  • Have an active edge selected (the edge around which faces will rotate)

Accessing the Tool

The Hinge Extrude tool can be accessed from multiple locations:

  • View3D → Mesh → Extrude → Hinge Extrude
  • View3D → Mesh → Faces → Hinge Extrude
  • Right-click context menu in Edit Mode

Basic Workflow

  1. Select the faces you want to extrude
  2. Select an edge and make it active (last selected edge becomes active)
  3. Enable both Edge and Face select modes
  4. Run the Hinge Extrude operator
  5. Adjust the Rotation Angle and Steps parameters as needed
Note: The active edge acts as the rotation axis. To make an edge active, select it last while holding Shift to keep your face selection.

Parameters

Rotation Angle

Type: Float (Angle)
Default: 30° (0.523599 radians)
Range: -360° to 360°

Controls the total angle of rotation for the extrusion. Positive values rotate in one direction, negative values rotate in the opposite direction. The rotation direction is automatically calculated based on the face normals to ensure intuitive behavior.

Steps

Type: Integer
Default: 1
Range: 1 to 100

Determines how many extrusion steps to perform. The total rotation angle is divided equally among all steps. Higher values create smoother curves but result in more geometry.

  • Steps = 1: Single extrusion at the full rotation angle
  • Steps = 4: Four extrusions, each rotated by 1/4 of the total angle
  • Steps = 12: Creates smooth circular geometry (useful for 90° or 180° rotations)

Tips & Examples

Creating Circular Extrusions

To create a complete circular extrusion (360°):

  • Set Rotation Angle to 360°
  • Set Steps to 12 or higher for smooth results
  • The tool will automatically merge overlapping vertices

Creating Hinged Geometry

For door hinges, lids, or mechanical parts:

  • Use smaller rotation angles (30° - 90°)
  • Use fewer steps (1-4) for hard-edged mechanical parts
  • The edge should be positioned where the hinge pivot would be

Understanding Rotation Direction

The tool automatically determines the rotation direction based on your face normals:

  • Positive angles rotate faces in the direction of their normals
  • If the rotation seems backwards, try negating the angle
  • Face orientation can be checked with Overlays → Face Orientation

Performance Considerations

  • Higher step counts create more geometry and may slow down viewport performance
  • The tool automatically removes duplicate vertices after extrusion
  • Complex selections with many faces will take longer to process

Common Issues

"No active edge selected"

  • Make sure an edge is the last element you selected
  • Try selecting faces first, then Shift+click an edge to make it active

"No faces selected"

  • Ensure you have at least one face selected
  • Both Edge and Face select modes must be enabled
