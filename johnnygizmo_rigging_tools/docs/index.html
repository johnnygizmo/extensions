Introduction

The JohnnyGizmo Rigging Tools add-on provides a set of tools to streamline the rigging process in Blender. It includes features for both mesh and armature objects, allowing for efficient parenting, bone manipulation, and constraint application.

Source code is available here.

Mesh Object Mode

Add Parent Bone

Assign a mesh to a specific bone. Select a mesh with no parent, run the tool, and choose the target armature and bone from the popup.

Change Parent Bone

Reassign a mesh to a different bone. Select a mesh that is already parented to a bone, run the tool, and choose the new target bone.

Parent Meshes to Nearest Bone

Automatically parent selected mesh objects to the nearest bone of the active armature. Useful for quickly attaching many loose parts to a skeleton.

  • Bone Point: Choose which part of the bone to measure distance from (Head, Tail, or Center).
  • Use Deform: Only consider bones with the 'Deform' property enabled.
  • Replace Parent: Force reparenting even if the object already has a parent.

Mesh Edit Mode

Mesh Bone Magnet

Snap a bone joint to the center of the current mesh selection. In Mesh Edit Mode (with the mesh parented to an armature), select vertices, edges, or faces, then use this tool to move a specific bone joint (Head or Tail) to that location.

Vertex Bone Assignment

Assign selected vertices to a specific bone. Great for weight painting and mesh deformation control. The closest bone will be chosen by default, but you can select another one as needed.

Add Bone At Selected

Create a new bone at the average position of your selected mesh elements. This tool lets you customize the name, direction, and length of the bone right after it's created.

Create Parent Armature

Generate a new armature rooted at the center of the current mesh selection. The mesh will be automatically parented to this new armature. Useful for starting a rig from scratch.

Armature Edit Mode

Armature Bone Magnet

Snap a specific bone joint to the average position of currently selected bone joints. Useful for aligning bones within the armature.

Bone Straightener

Straighten selected bones by aligning them to a global axis.

Align Bone to Face Normal

Align the active bone to the normal of a selected face on a mesh object. Requires one armature and one mesh selected.

  • Preserve Bone Length: Maintain the original length of the bone.
  • Align to Active Edge: Align the bone's secondary axis (roll) to the active edge on the mesh.
  • Edge Align Axis: Choose which axis (X or Z) to align to the edge.
  • Flip Roll 180Â°: Flip the bone roll by 180 degrees.

Align Connected Bone Children

Align all connected child bones to the direction of the selected parent bone.

  • Copy Roll: Apply the parent's roll to all aligned children.
  • Copy Length: Set all aligned children to the same length as the parent.

Bone Chain Rename

Batch rename a chain of bones with standard naming conventions (Side, Type, Numbering). Works in both Edit and Pose modes.

Armature Pose Mode

Bone Doctor

Perform a health check and automated cleanup on your armature.

  • Disable Deform on IK Targets: Unchecks 'use_deform' for bones used as IK targets.
  • Disable/Enable Deform on DEF/Non-DEF: Enforces deformation settings based on naming conventions (DEF_*).
  • Move to Collections: Automatically organizes bones into DEF, MCH, and CTRL collections based on their names.
  • Clean Symmetry Naming: Standardizes suffixes (e.g., converts _L to .L) for Blender symmetry to work correctly.
  • Apply Armature Transforms: Applies object-level scale and rotation to the armature data.
  • Generate Report: Creates a text block detailing potential issues like missing targets, zero-length chains, or unassigned bones.

Create Widget Driver

Create complex drivers to control shape keys using bone transforms.

  • Control Transform: Use Location, Rotation, or Scale of the bone to drive the key.
  • Range Mapping: Map a specific range of bone movement (e.g., 0 to 1 meter, or 0 to 90 degrees) to the shape key value (0 to 1).
  • Constraints: Automatically add limit constraints to the bone to prevent it from moving outside the defined control range.
  • Bone Collection: Assign the control bone to a specific collection (e.g., "CTL") for organization.

Advanced IK (IK+)

quickly add an IK constraint with a predefined setup.

  • Chain Length: Set the number of bones in the IK chain.
  • Pole Target: Automatically assign a Pole Target object and bone.
  • Weights: Fine-tune Position and Rotation weights immediately.

Standard Constraint Tools

Quickly apply standard constraints: Damped Track, Stretch To, Locked Track, and Track To.

  • One Bone Selected: Adds the constraint to the selected bone (requires setting target manually if not prompted).
  • Two Bones Selected: The active bone becomes the owner, and the inactive bone becomes the target.
