Introduction

The JohnnyGizmo Rigging Tools add-on provides a set of tools to streamline the rigging process in Blender. It includes features for both mesh and armature objects, allowing for efficient parenting, bone manipulation, and constraint application.

Source code is available here.

Mesh Object Mode

Add Parent Bone

In Mesh Object Mode, you can assign a single mesh to a bone using the "Add Parent Bone" tool. You'll be presented with an object picker to select the armature and bone. Select a mesh object with no parent to make this tool available.

Change Parent Bone

The "Change Parent Bone" tool lets you change the parent bone of a mesh. Just like the Add Parent tool, you select the armature and new bone. This tool appears when a mesh with an existing parent is selected.

Mesh Edit Mode

Mesh Bone Magnet

Snap a bone joint to a mesh selection for quick skeleton alignment. In edit mode of a mesh parented to an armature, select one or more vertices, edges, or faces, then use the "Mesh Bone Magnet" tool to move the joint to that location.

Vertex Bone Assignment

Assign selected vertices to a specific bone. Great for weight painting and mesh deformation control. The closest bone will be chosen by default, but you can select another one as needed.

Add Bone At Selected

Create a new bone at the average position of your selected mesh elements. This tool lets you customize the name, direction, and length of the bone right after it's created.

Create Parent Armature

In Mesh Edit Mode, if the mesh does not have a parent, you can create a parent armature for your mesh. The new armature will be created at the average position of the selected mesh elements and the mesh will be parented to the new armature.

Armature Edit Mode

Armature Bone Magnet

With one or more bone joints selected, snap another bone joint to the average position of the selected bones.

Bone Straightener

Straighten selected bones by aligning them to a global axis.

Armature Pose Mode

Constraint Tools

In pose mode if one or two bones are selected, you can use the "Constraint Tools" to add constraints IK, Damped Track, Stretch To, and Locked Track. The options for the constraints will appear for faster assigning. If one bone is selected, the constraint will be applied to that bone, and all other options will need to be set. If two bones are selected, the inactive bone will be the target and the active bone will be the owner of the constraint.
