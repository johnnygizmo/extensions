Introduction

The JohnnyGizmo Align Tools add-on provides a collection of precision alignment utilities for Blender. These tools help streamline the process of aligning faces, positioning origins, and working with mesh geometry and armatures. The tools are designed to save time and improve accuracy when building and rigging 3D models.

Mesh Edit Mode

Align Face to Second Active Face

Move and rotate one mesh object so its selected face(s) align perfectly to another mesh object's selected face(s). This tool is ideal for precisely connecting mesh parts or aligning surfaces.

Usage:
  1. Select two mesh objects and enter Edit Mode (Tab)
  2. Select an active face or set of faces on each mesh
  3. Run the operator from the Mesh menu or right-click context menu

Options:

  • Move Active: Choose which of the two objects to move. When enabled, moves the active object; when disabled, moves the inactive object.
  • Flip Normal Direction: Controls whether the normals point in the same direction or opposite directions. When disabled (default), normals become opposite for face-to-face alignment.
  • Twist Angle: Rotate the moved object around the face normal (in degrees). Use this to fine-tune rotational alignment.
  • Offset Distance: Translate the moved object along the target normal after alignment. Positive values move away from the target face, negative values move into it.
  • Match Edge Rotation: When enabled, aligns the active edges from both meshes to match edge directions. This requires selecting both faces and an edge on each mesh (mixed face/edge selection).
  • Flip Edge Direction: Adds 180 degrees to the edge rotation (only works when Match Edge Rotation is enabled).
Edge Alignment Tip: For precise edge-to-edge alignment, select faces plus one active edge on each mesh, then enable "Match Edge Rotation". The tool will align the edges perpendicular to the face normal.

Object Mode

Set Origin to Bounding Box Position

Position the object's origin at any corner, edge, or face of its bounding box. This tool is perfect for setting pivot points at specific locations like the bottom center for characters, or corners for architectural elements.

Usage:
  1. Select a mesh object in Object Mode
  2. Run the operator from the Object menu or right-click context menu
  3. Choose options for Vertical, Horizontal, and Depth positioning

Options:

  • Vertical: Set the vertical (Z-axis) position
    • Top - Top of bounding box
    • Center - Center of bounding box (vertical)
    • Bottom - Bottom of bounding box (default)
  • Horizontal: Set the horizontal (X-axis) position
    • Left - Left side of bounding box
    • Center - Center of bounding box (horizontal, default)
    • Right - Right side of bounding box
  • Depth: Set the depth (Y-axis) position
    • Front - Front of bounding box
    • Center - Center of bounding box (depth, default)
    • Back - Back of bounding box
Common Use Cases:
  • Character Origin: Set to Bottom-Center-Center for proper pivot point at the feet
  • Wheel Origin: Set to Center-Center-Center for rotation around the wheel's center
  • Wall Origin: Set to Bottom-Left-Back for corner-based placement

Tips and Tricks

Face Alignment Workflow

For best results when aligning faces:

  • Use area-weighted face selection (multiple faces) for more stable alignment
  • Start with basic alignment, then fine-tune with Twist Angle and Offset Distance
  • Use Match Edge Rotation for mechanical parts that need precise angular alignment
  • Remember that "Flip Normal Direction" controls whether surfaces face each other or the same direction

Origin Positioning Workflow

The bounding box origin tool is most useful when:

  • Setting up character rigs (Bottom-Center for feet placement)
  • Preparing objects for array modifiers (corner origins for building blocks)
  • Creating mechanical assemblies (center origins for rotating parts)
  • Organizing architectural elements (corner origins for walls and floors)

Menu Locations

Align Face to Face

  • Edit Mode → Mesh → Transform → Align Face To Face
  • Edit Mode → Face → Align Face To Face
  • Edit Mode → Right-click context menu → Align Face To Face

Set Origin to Bounding Box Position

  • Object Mode → Object → Set Origin to Bounding Box Position
  • Object Mode → Right-click context menu → Set Origin to Bounding Box Position
